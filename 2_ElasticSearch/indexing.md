# Search Processing
Tokenization – breaking text into searchable terms
Analyzers – transforming terms to improve matching
Scoring – ranking results by relevance

# Lucene
Apache Lucene is a high-performance, full-featured, open-source search engine library written entirely in Java. It is widely considered the de facto standard for implementing search capabilities in applications, serving as the foundational technology for major search platforms like Elasticsearch, Apache Solr, and OpenSearch.

# ElasticSearch
Elasticsearch is a distributed, RESTful search and analytics engine designed for speed, scale, and handling diverse data types. Built on Apache Lucene, it functions as the central component of the Elastic Stack (formerly ELK Stack), enabling near real-time storage, retrieval, and analysis of massive datasets.
